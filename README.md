# Premium Calculator

A sleek, visually stunning calculator application featuring a modern glassmorphism design.

## Features
- **Rich Aesthetics**: Dark mode with glassmorphism effects (blur background) and neon accents.
- **Responsive Layout**: Works on different screen sizes.
- **Smooth Animations**: Hover effects and floating animations for a premium feel.
- **Full Operations**: Addition, subtraction, multiplication, division, percentage, and sign toggling.

## Tech Stack
- Semantic HTML5
- Vanilla CSS3 (Custom properties, backdrop-filter)
- Vanilla JavaScript

## How to Run

To run the application locally, you can use the Ruby HTTP server (as tested on this environment):

```bash
ruby -run -e httpd . -p 8080
```

Then open your browser and navigate to `http://localhost:8080`.
